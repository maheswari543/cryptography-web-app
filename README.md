# 🛡 Cryptography Web App

## 📌 Introduction
This project is a web-based application that demonstrates various cryptographic algorithms. It allows users to perform encryption and decryption using different techniques.

## 🚀 Features
- Encryption & Decryption modules
- Algorithms:
  - Caesar Cipher
  - Substitution Cipher
  - Hill Cipher
  - DES
  - Blowfish
  - AES
  - RSA
  - Diffie-Hellman
- Copy to Clipboard option
- Detailed Notes section
- Interactive Quiz with Score

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript

## ▶️ How to Run
1. Download or clone the project
2. Open `dashboard.html` in browser

## 🎯 Objective
- Understand cryptography concepts
- Implement encryption algorithms
- Learn symmetric & asymmetric techniques

## 📊 Output
User can encrypt/decrypt text and test knowledge using quiz.

## 📚 Conclusion
This project helps in learning cryptography concepts in an interactive way.
