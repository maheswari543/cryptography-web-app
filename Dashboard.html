📘 Detailed Cryptography Notes

What is Cryptography?
Cryptography is the technique of securing communication by converting readable data (Plain Text) into an unreadable format (Cipher Text).

Importance:
• Protects data
• Ensures privacy
• Used in banking and security systems

Types:
1. Classical → Caesar, Substitution, Hill
2. Symmetric → AES, DES, Blowfish
3. Asymmetric → RSA, Diffie-Hellman

Process:
Plain Text → Encryption → Cipher Text → Decryption → Original Text

Algorithms:

  • Caesar → Shift letters
  • Substitution → Replace letters
  • Hill → Matrix method
  • DES → Old encryption
  • Blowfish → Fast secure
  • AES → Modern standard
  • RSA → Public/private keys
  • Diffie-Hellman → Key exchange

Applications:
Banking, passwords, communication security.
